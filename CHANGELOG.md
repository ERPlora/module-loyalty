## [1.0.0] - 2025-12-28

### Added

- Initial release of Loyalty module
- **Core Features**
  - Points earning on purchases
  - Points redemption
  - Customer balance tracking
  - Points expiration

- **Models**
  - `LoyaltyProgram`: Program configuration
  - `LoyaltyMember`: Customer membership
  - `PointsTransaction`: Points history
  - `Reward`: Redeemable rewards

- **Internationalization**
  - English translations (base)
  - Spanish translations
